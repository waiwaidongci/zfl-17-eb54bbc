# 胶片分镜条核对台

一个纯前端本地应用，用来整理纸质放映胶片片段、拖拽调整放映顺序、统计总时长、筛选颜色偏移并生成试映前破损提醒。

直接打开 `index.html` 即可使用，也可以在本目录启动静态服务访问。

## 功能概览

- **录入片段**：填写片段编号、时长、颜色偏移、破损情况、缩略图和备注
- **片段模板**：保存常用配置为模板，录入时一键套用
- **放映顺序**：拖拽或上下按钮调整片段顺序
- **统计筛选**：按颜色偏移筛选、按关键词搜索、实时显示总时长和破损数
- **试映核对**：自动列出存在颜色偏移或破损的片段供放映前核对
- **导出清单**：将完整清单导出为文本文件
- **批量导入**：粘贴或上传 CSV 文本，预览确认后批量写入放映清单

## 批量导入使用说明

### CSV 格式

CSV 文件每行一条片段，共 **5 列**（最后 1 列「备注」可选），列顺序如下：

```
片段编号, 秒数, 颜色偏移, 破损情况, 备注
```

**可选值范围：**
- **颜色偏移**：`正常`、`偏红`、`偏青`、`偏黄`、`褪色`
- **破损情况**：`完好`、`轻微划痕`、`齿孔破损`、`接片松动`、`需跳过`

### 可复制的 CSV 示例

```csv
片段编号,秒数,颜色偏移,破损情况,备注
A-001,18,正常,完好,开场街景，节奏平稳
A-002,12,偏红,轻微划痕,人物近景，左侧有划痕
A-003,14,褪色,接片松动,接片位置靠近段尾
A-004,8,正常,完好,过渡镜头
A-005,20,偏青,齿孔破损,齿孔破损，放映前检查
A-006,10,偏黄,完好,暖色调场景
A-007,15,正常,需跳过,此片段需跳过，暂不使用
A-008,16,褪色,轻微划痕,旧片，整体褪色严重
```

### 导入流程

1. 点击工具栏的「批量导入」按钮
2. 选择以下任意一种方式输入 CSV：
   - **粘贴文本**：直接将 CSV 内容粘贴到文本框中
   - **上传文件**：点击「上传 CSV 文件」选择 `.csv` 或 `.txt` 文件
3. 点击「解析预览」按钮，系统将自动解析并显示预览表
4. 检查预览表中的状态：
   - ✓ **有效**：数据格式正确，可以导入
   - ⚠ **重复**：编号在 CSV 内重复或与当前清单冲突，将跳过
   - ✗ **错误**：数据格式有误，将跳过
5. 确认无误后点击「确认导入有效行」，有效片段将被添加到当前放映清单

### 导入注意事项

- **列数要求**：每行至少 4 列，第 5 列（备注）可选
- **秒数要求**：必须为正整数（如 `12`，不能是 `abc`、`-5` 或 `0`）
- **编号唯一性**：相同编号的片段只能导入一条，系统会自动检测并标记重复
- **可选值严格匹配**：颜色偏移和破损情况必须从可选值中选择，否则视为错误
- **第一行表头**：如果第一行是表头（如「片段编号,秒数,...」），系统会自动跳过
- **空白行**：空白行会被自动忽略
- **引号支持**：如果备注中包含逗号，可用双引号包裹（如 `A-009,10,正常,完好,"画面抖动,需留意"`）
- **导入追加**：导入的片段会追加到当前清单末尾，不会覆盖现有内容
- **确认机制**：导入前会弹出确认对话框，显示即将导入的有效行数

## 片段模板使用说明

### 创建模板

1. 在「片段模板」面板中填写：
   - **模板名称**（必填）：便于识别的名称，如「常规完好片」「破损近景片」
   - **默认时长秒数**：该类片段的默认时长
   - **颜色偏移**：正常 / 偏红 / 偏青 / 偏黄 / 褪色
   - **破损情况**：完好 / 轻微划痕 / 齿孔破损 / 接片松动 / 需跳过
   - **备注前缀**：可选，例如「【旧片】」「【留意】」
2. 点击「保存为新模板」完成创建

### 套用模板

模板提供两种套用方式：

1. **录入面板快速选择**：在「录入片段」面板顶部的下拉框选择模板，点击「套用」或直接选择即可将配置填入表单
2. **模板卡片一键套用**：在「片段模板」面板中，点击任意模板卡片上的「一键套用」按钮

套用模板时：
- 时长、颜色偏移、破损情况会直接覆盖当前表单值
- 备注前缀会追加到备注内容开头（如果已有内容且未以该前缀开头）

### 删除模板

点击模板卡片右上角的「×」按钮即可删除对应模板。

### 本地存储

所有模板数据与片段清单一同保存在浏览器本地（localStorage），刷新或关闭页面后不会丢失。
